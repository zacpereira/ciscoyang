Schema Definitions

event-manager_schema:

Event manager configuration

scheduler-script: object

scheduler classs type

thread-classes: object

scheduler thread classs

thread-class: object[]

scheduler classs type argument

object
thread-class-name: string

Name of the global variable

num-threads: integer

number of scheduler threads

range:"1..5"
directory-user-policy: string

Set event manager user policy directory

refresh-time: integer

Set refresh time (in seconds) for policy username's AAA taskmap

range:"10..4294967295" units:"second" default:"1800"
environments: object

Set an event manager global variable for event manager policies

environment: object[]

Name of the global variable

object
environment-name: string

Name of the global variable

environment-value: string

Value of the global variable

schedule-suspend: boolean

Enable suspend policy scheduling

default:
policies: object

Register an event manager policy

policy: object[]

Name of the policy file

object
username: string

A configured username

checksum-type: string sha-1, md5

Specify Embedded Event Manager policy checksum

policy-type: string system, user

Event manager type of this policy

policy-security-level: string rsa-2048, trust

Event Manager policy security Level

policy-name: string

Name of the policy file

persist-time: string

Time of validity (in seconds) for cached AAA taskmap of username (default is 3600)

check-sum-value: string

CheckSum Value

policy-security-mode: string cisco, trust

Specify Embedded Event Manager policy security mode

directory-user-library: string

Path of the user policy library directory

Example
{
  "scheduler-script": {
    "thread-classes": {
      "thread-class": [
        {
          "thread-class-name": "string",
          "num-threads": "integer"
        }
      ]
    }
  },
  "directory-user-policy": "string",
  "refresh-time": "integer",
  "environments": {
    "environment": [
      {
        "environment-name": "string",
        "environment-value": "string"
      }
    ]
  },
  "schedule-suspend": "boolean",
  "policies": {
    "policy": [
      {
        "username": "string",

        "persist-time": "string",
