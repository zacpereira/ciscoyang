PPP-EA-INTERFACE-INFO:

PPP EA interface information

is-ipv6cp-running: boolean

TRUE if IPV6CP is running in the dataplane for the interface

forwarding-enabled: boolean

Forwarding State

peer-magic: integer

Peer magic number

is-lcp-running: boolean

TRUE if LCP is running in the dataplane for the interface

TRUE if IPCP is running in the dataplane for the interface

Echo-Request interval

Local interface MTU

IPCP VRF Table ID

IPv6CP VRF Table ID

L2 adjacency state

MPLS adjacency state

Echo-Request retry count

l2-provisioned: boolean

L2 Provisioned State

IPv6 adjacency state

l2ip-interworking-adjacency-state: string ppp-ea-adj-state-not-installed, ppp-ea-adj-state-installed

L2 IP Interworking adjacency state

interface-adjacency-state: string ppp-ea-adj-state-not-installed, ppp-ea-adj-state-installed

Interface adjacency state

multilink-interface: string

Multilink interface that this interface is a member of, if any

interface: string

Interface

is-multilink-bundle: boolean

TRUE if this is a Multilink bundle interface

l2-tunnel-enabled: boolean

L2 Tunnel State

local-mrru: integer

Local MRRU

l2ip-interworking-enabled: boolean

L2 IP Interworking State

lac-adjacency-state: string ppp-ea-adj-state-not-installed, ppp-ea-adj-state-installed

LAC adjacency state

local-magic: integer

Local magic number

is-mplscp-running: boolean

TRUE if MPLSCP is running in the dataplane for the interface
