Schema Definitions

ipv6-configuration_schema:

IPv6 Configuration Data

ipv6-pmtu-time-out: integer

Configure IPv6 Path MTU timeout value in minutes

range:"1..15" units:"minute"
ipv6icmp: object

Configure IPv6 ICMP parameters

error-interval: integer

Interval between tokens in milliseconds

range:"0..2147483647" units:"millisecond"
bucket-size: integer

Bucket size

range:"1..200" default:"10"
ipv6-pmtu-enable: boolean

TRUE if enabled, FALSE if disabled

default: false
ipv6-source-route: boolean

TRUE if enabled, FALSE if disabled

default: true
ipv6-assembler: object

IPv6 fragmented packet assembler

max-packets: integer

Maxinum packets allowed in assembly queues (in percent)

range:"1..50" units:"percentage"
timeout: integer

Number of seconds an assembly queue will hold before timeout

range:"1..120" units:"second"
ipv6-hop-limit: integer

Configure IPv6 hop count limit

range:"1..255"
Example
{
  "ipv6-pmtu-time-out": "integer",
  "ipv6icmp": {
    "error-interval": "integer",
    "bucket-size": "integer"
  },
  "ipv6-pmtu-enable": "boolean",
  "ipv6-source-route": "boolean",
  "ipv6-assembler": {
    "max-packets": "integer",
    "timeout": "integer"
  },
  "ipv6-hop-limit": "integer"
}
