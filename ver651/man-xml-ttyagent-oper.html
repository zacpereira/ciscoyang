Schema Definitions

XR-XML-SESSION-INFO:

XML SESSION INFO

username: string

Username

config-session-id: string

Config session ID

admin-config-session-id: string

Admin config session ID

vrf-name: string

VRF name

client-port: integer

client's port

state: string idle, busy

state of the session idle/busy

client-address: string

ip address of the client

elapsed-time: integer

Elapsed time(seconds) since a session is created

alarm-notification: string registered, not-registered

is the session registered for alarm notifications

last-state-change: integer

Time(seconds) since last session state change happened

start-time: integer

session start time in seconds since the Unix Epoch
