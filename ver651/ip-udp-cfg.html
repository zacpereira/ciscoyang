receive-q: integer

range:"40..800"
num-thread: object

UDP InQueue and OutQueue threads

Example
{
  "directory": {
    "max-file-size-files": "integer",
    "directoryname": "string",
    "max-udp-debug-files": "integer"
  },
  "receive-q": "integer",
  "num-thread": {
    "udp-in-q-threads": "integer",
    "udp-out-q-threads": "integer"
  }
}

num-thread_schema:

UDP InQueue and OutQueue threads

udp-in-q-threads: integer

InQ Threads

range:"1..16"
udp-out-q-threads: integer

OutQ Threads

range:"1..16"
Example
{
  "udp-in-q-threads": "integer",
  "udp-out-q-threads": "integer"
}

directory_schema:

UDP directory details

max-file-size-files: integer

Set size of debug files in bytes

range:"1024..4294967295" units:"byte"
directoryname: string

Directory name

max-udp-debug-files: integer

Set number of Debug files

range:"1..5000" default:"256"
Example
{
  "max-file-size-files": "integer",
  "directoryname": "string",
  "max-udp-debug-files": "integer"
}
