ip-raw_schema:

Global IP RAW configuration

directory: object

RAW directory details

receive-q: integer

RAW receive Queue Size

range:"40..800"
num-thread: object

RAW InQueue and OutQueue threads

Example
{
  "directory": {
    "max-raw-debug-files": "integer",
    "max-file-size-files": "integer",
    "directoryname": "string"
  },
  "receive-q": "integer",
  "num-thread": {
    "raw-in-q-threads": "integer",
    "raw-out-q-threads": "integer"
  }
}

num-thread_schema:

RAW InQueue and OutQueue threads

raw-in-q-threads: integer

InQ Threads

range:"1..16"
raw-out-q-threads: integer

OutQ Threads

range:"1..16"
Example
{
  "raw-in-q-threads": "integer",
  "raw-out-q-threads": "integer"
}

directory_schema:

RAW directory details

max-raw-debug-files: integer

Set number of Debug files

range:"1..18000" default:"256"
max-file-size-files: integer

Set size of debug files in bytes

range:"1024..4294967295" units:"byte"
directoryname: string

Directory name

Example
{
  "max-raw-debug-files": "integer",
  "max-file-size-files": "integer",
  "directoryname": "string"
