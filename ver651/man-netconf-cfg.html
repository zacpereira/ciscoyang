session_schema:

Session settings

absolute-timeout: integer

Absolute timeout in minutes

range:"1..1440" units:"minute"
idle-timeout: integer

Non-active session lifetime

range:"1..1440" units:"minute"
limit: integer

Count of allowable concurrent netconf-yang sessions

range:"1..50" default:"50"
Example
{
  "absolute-timeout": "integer",
  "idle-timeout": "integer",
  "limit": "integer"
