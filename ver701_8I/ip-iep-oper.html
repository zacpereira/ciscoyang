Schema Definitions

IEP-IP-ADDRESS-ENTRY:

IP address configured in the explicit path

index: integer

Index number at which the path entry is inserted or modified

address-type: string next, exclude, exclude-srlg

Specifies the address type

hop-type: string strict, loose

Specifies the next unicast address in the path as a strict or loose hop

mpls-label: integer

MPLS label

address: string

IPv4 unicast address

if-index: integer

Interface Index of the path
